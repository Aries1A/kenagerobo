# kenagerobo

## 概要
KenageRoboは、「[東京大学制作展 2019 ああ言えば、こう言う。こう言えば、どう言う？](http://www.iiiexhibition.com/)」にて制作・展示した作品です。

思わず応援したくなるロボットに出会ったことはありますか。

KenageRoboは強化学習をするロボットの「健気さ」をテーマとした作品です。

ロボットに与えられた課題は「制限時間内にゴールまで進むこと」。

何度失敗してもひたむきに挑戦し続ける球ロボットの成長日記です。
